# FinHouse.ai mini site (v3 - partner ikonları + sektör görselleri)

## Çalıştırma
ZIP'i açın, klasör içine girin.

### Python
python3 -m http.server 8080

### Node
npx serve .

Tarayıcı:
http://localhost:8080

## Tema
Sağ üstte 'Koyu/Açık' butonu.
Tema değişince logo da otomatik değişir.

## Not
Bu paket, görsel/ikonları projeye gömülü olarak getirir (assets/).
